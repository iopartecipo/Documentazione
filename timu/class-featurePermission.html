Class featurePermission

This Class will have all static variables. $features variable is used to define all features of website where composite permission check is to be applied. All features defined in $features varaible must be explictly define as a sepreate variable and it should define the permission valid for that feature.

checkPermission function will automatically ready updates from this file.
